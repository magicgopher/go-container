# 使用go操作容器

## 概述

本仓库系统记录了**使用 Go 语言操作 Docker 和 Podman 容器** 的完整流程、示例代码、最佳实践以及踩坑笔记。

核心特点：
- **无缝兼容**：Podman 原生支持 Docker API，因此大部分代码只需改一行配置即可在 Podman 上运行
- **双客户端支持**：同时展示官方 `docker/client` 和 Podman 原生 `bindings`
- **面向记录**：每个示例都配有详细中文注释 + 运行截图说明，适合学习和快速查阅
- **可扩展**：未来可轻松扩展到 containerd、CRI-O 等

无论你是想写自己的容器编排工具、CI/CD 插件，还是单纯学习容器底层操作，这里都能找到现成可运行的代码。
